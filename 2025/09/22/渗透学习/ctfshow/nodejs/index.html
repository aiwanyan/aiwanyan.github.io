语法函数:

1.eval()函数:

特性eval()child_process.exec() / execSync() 等
功能执行 JS 表达式/代码字符串执行系统命令，调用 shell
执行环境JavaScript 运行时操作系统 shell（bash/cmd）
数据类型仅 JS 数据（字符串、对象等）任意系统命令输出（Buffer/String）
返回值JS 表达式的结果子进程的输出（stdout）
安全风险高 —— 代码注入高 —— 命令注入
CTF利用点SSTI、模板注入等可拼接 JS 表达式文件上传后配合执行；命令拼接
防御建议禁用 eval；使用沙箱或 vm 模块参数白名单；转义特殊字符

参数说明
view模板名称（不带扩展名），如 index

callback（可选）如果提供，则不会自动发送响应，而是返回 HTML 字符串
