2.复制文件命令:

1.cp命令:

命令说明
cp a.txt b.txt把 a.txt 复制成 b.txt
cp a.txt /tmp/把 a.txt 复制到 /tmp/ 目录
cp -r dir1 dir2递归复制目录（必须加 -r）
cp -f a b强制覆盖目标文件
cp -u a b仅当源文件比目标新时才复制（常用于备份）

2.tee命令:

命令说明
cp a.txt b.txt把 a.txt 复制成 b.txt
cp a.txt /tmp/把 a.txt 复制到 /tmp/ 目录
cp -r dir1 dir2递归复制目录（必须加 -r 选项）
cp -f a b强制覆盖目标文件，无论目标文件是否存在
cp -u a b仅当源文件比目标文件更新时才复制（常用于备份）
cp -i a b在覆盖目标文件前提示用户确认
cp -v a b显示详细的复制过程，列出正在复制的文件
cp -a dir1 dir2复制目录及其内容，保持文件的原始属性（包括符号链接）
cp -l a b创建目标文件的硬链接，而非复制

7.创建一个文件:

命令示例作用特点
touch file.txt创建空文件文件已存在时更新时间戳
echo "hello" > file.txt创建并写入内容会覆盖已有文件内容
cat > file.txt （输入内容后按 Ctrl+D）手动输入内容创建文件适合一次性写多行
printf "flag{123}\n" > flag.txt创建并写入格式化内容可用于生成 CTF flag 等结构化内容
> empty.txt创建空文件文件已存在则清空内容
dd if=/dev/zero of=bigfile.bin bs=1M count=10创建指定大小文件常用于测试、磁盘填充
vi file.php nano file.txt打开编辑器创建写完保存即可生成文件

8.查看进程;

命令说明示例
ps aux显示所有用户的所有进程`ps aux
top / htop实时显示进程动态信息top（htop更美观）
pidof <name>查找某个程序的 PIDpidof clash
pgrep <pattern>进程名匹配并返回 PIDpgrep -fl clash
lsof -p <pid>查看某进程打开的文件lsof -p 1234
netstat -tunlp查看进程监听端口`netstat -tunlp
ss -tulnp更快的 netstat 替代`ss -tulnp
strace -p <pid>跟踪某进程系统调用strace -p 1234

9.查看端口:

命令作用示例
netstat -tunlp查看所有监听端口 + 对应进程`netstat -tunlp
ss -tulnp更快的 netstat 替代`ss -tulnp
lsof -i:<端口>查看端口对应的进程lsof -i:7890
fuser <端口>/tcp快速查看哪个PID占用了端口fuser 8080/tcp
cat /proc/net/tcp直接读取系统 TCP 表可写脚本查所有端口状态
nmap 127.0.0.1 -p-扫描本地开放端口本地横向扫描、提权辅助

1.参数相关:

参数含义作用
-tTCP仅显示 TCP 连接
-uUDP仅显示 UDP 连接
-lListen仅显示监听状态（如服务器在等待连接）

-pProcess显示哪个进程（PID/程序名）占用此连接，需要 root 权限

11.ls命令

参数含义说明
-l以长格式列出文件详细信息（权限、用户、大小、修改时间等）
-a显示所有文件，包括以 . 开头的隐藏文件
-h与 -l 配合使用，显示人类可读的文件大小（如 1K，234M）
-S按文件大小排序，默认是从大到小
-r反向排序（与 -t、-S 等配合使用时效果明显）
-t按时间排序，最近修改的文件排在前面
-R递归列出子目录内容
-d如果后面跟的是目录，只显示目录本身而不是其内容
-F在文件名后附上符号（如 / 表示目录，* 表示可执行）
-i显示文件的 inode 编号
-n类似 -l，但显示 UID 和 GID 而非用户名和组名
--color=auto给文件名上色（目录蓝色、可执行绿色等），通常默认启用
--time=ctime使用变更时间（ctime）排序，而不是默认的修改时间（mtime）
-1每个文件单独一行显示（默认多个列显示时使用）

ls常用组合

命令用法说明
ls -alh列出所有文件 + 详细信息 + 可读大小
ls -lhS按大小排序显示文件
ls -lt按修改时间排序
ls -ltr按时间升序排列（旧文件在前）
ls -R查看当前目录及所有子目录内容

12.head命令:

参数含义示例
-n N显示前 N 行（默认是 10 行）head -n 5 file.txt
-c N显示前 N 个字节head -c 20 file.txt

14.tail 命令:

参数含义示例
-n N显示最后 N 行（默认 10 行）tail -n 5 error.log
-c N显示最后 N 个字节tail -c 100 file.txt
-f实时输出追加的内容（日志监控）tail -f /var/log/syslog
+N从第 N 行开始显示（常用于搭配 tail -n +2）tail -n +2 file.txt

权限类型命令说明

更改权限chmod 命令rwx 权限设置（读/写/执行）
加入组usermod -aG 组名 用户加用户到指定用户组
设置 ACLsetfacl -m精细设置用户/组的访问权限

查看用户权限;

命令用途示例输出解释
whoami查看当前用户名whoami如：alice
id查看 UID、GID、所属组iduid=1001(alice) gid=1001(alice) groups=1001(alice),100(users)
groups显示所属的所有组groups如：alice users
echo $PATH查看命令搜索路径echo $PATH可执行命令所在目录
ls -l /路径/文件查看文件权限ls -l /etc/passwd-rw-r--r-- 表示所有人可读
namei -l /路径/文件逐级查看路径权限namei -l /etc/passwd检查某级目录权限是否阻断
getfacl /路径/文件查看文件的 ACL 权限（如果启用）getfacl myfile显示具体用户的附加权限
test -x 命令路径判断某程序是否可执行test -x /usr/bin/python3 && echo "可执行"返回 “可执行” 或 “不可执行”
`cat /etc/passwdgrep ^$(whoami)`查看当前用户的 shell、主目录等`cat /etc/passwd
段位字符含义
文件类型-普通文件（d 是目录，l 是符号链接）
所属用户权限rw-root 用户：可读、可写
所属组权限r--root 组：可读（但不可写或执行）
其他用户权限r--所有其他用户：可读（不可写或执行）
