curl命令:

类型示例命令说明
GET 请求curl http://localhost:8080/hello发起简单的 GET 请求
POST 表单curl -X POST -d "name=wan&age=18" http://localhost:8080/form-d 表示 form 表单数据，默认 application/x-www-form-urlencoded
POST JSONcurl -X POST -H "Content-Type: application/json" -d '{"name":"wan","age":18}' http://localhost:8080/json-H 指定请求头，发送 JSON 数据
添加请求头curl -H "Authorization: Bearer TOKEN" http://localhost:8080/api模拟登录认证等，常用于 JWT
携带 Cookiecurl --cookie "sessionid=abc123" http://localhost:8080/auth用于模拟浏览器 session
响应输出控制curl -i（输出响应头）curl -v（详细调试信息）-i 显示响应头，-v 打印过程
上传文件curl -F "file=@test.jpg" http://localhost:8080/upload用于 multipart/form-data 文件上传
设置代理curl -x 127.0.0.1:7890 http://example.com使用 HTTP 代理发送请求
保存响应curl -o out.txt http://localhost:8080/data将响应保存至文件中
禁止证书校验curl -k https://localhost:8443用于测试自签名 HTTPS 接口

7.ip地址的各种形式:

类型表示方法示例说明
✅ 标准十进制a.b.c.d127.0.0.1最常见
✅ 十进制整数单个十进制数2130706433127×256³ + 0×256² + 0×256 + 1
✅ 八进制0177.0.0.010177.0.0.01八进制：0177=127，01=1
✅ 十六进制0x7f.0x0.0x0.0x10x7f.0x0.0x0.0x10x7f=127，支持绕过
✅ 整体十六进制0x7f000001127.0.0.1一整个 IP 作为 hex
✅ 整体混合表示127.1等价 127.0.0.1后两段自动补全为0
✅ 域名解析localhostlocalhostDNS解析成127.0.0.1
✅ DNS跳转evil.ceye.io解析为内网 IP配合 DNS rebinding
✅ 数组式分段127.0.1= 127.0.0.1缺省补0，浏览器兼容
✅ 十六进制分段混用0x7f.1127.0.0.1分段可混合进制
表达形式对应 IP
127.0.0.1127.0.0.1
127.1127.0.0.1
0x7f.0.0.1127.0.0.1
0x7f000001127.0.0.1
0177.0.0.01127.0.0.1

[::1]127.0.0.1 (IPv6 回环)
