2.re模块;

1.基础用法:

函数用途示例
re.search()匹配一个符合的re.search(r'flag{.*}', s)
re.findall()返回所有匹配内容re.findall(r'\d+', s)
re.sub()替换匹配项re.sub(r'sleep\([^)]+\)', '1', payload)
re.match()匹配开头（从头匹配）re.match(r'^GET', line)
re.split()正则分割字符串re.split(r'\s+', s)
re.compile()编译正则对象r = re.compile(r'sleep\(\d+\)')

2.正则表达式;

表达式含义示例
.任意字符（不包括换行）a.b 匹配 acb, arb
*重复0次或多次.* 匹配任意字符串
+重复1次或多次\d+ 匹配 123
?0或1次（或非贪婪）.*? 匹配最短字符串
[...]字符集[0-9] 匹配数字
[^...]非字符集[^A-Z] 非大写字母
\d数字（等价于 [0-9]）\d{4} 匹配4位数
\w单词字符（字母数字下划线）\w+ 匹配变量名
\s空白字符\s+ 匹配任意空格、换行
^行首^flag 以 flag 开头
$行尾abc$ 以 abc 结尾
``或
简写旗标完整常量黑客视角用途 / 功能说明常见攻击场景
re.Ire.IGNORECASE忽略大小写，匹配更宽松绕过大小写 WAF 关键词过滤 (SeLeCt, UnIoN)
re.Sre.DOTALL让 . 匹配换行，跨行抓取大块 Payload抓多行注入语句、跨标签 XSS 片段
re.Mre.MULTILINE^/$ 从行首尾而非全文首尾匹配分析多行日志，一行一行定位恶意请求
re.Xre.VERBOSE正则可加空格、换行、注释，团队协作易读写复杂匹配规则（SQLMap 策略等）时避免维护地狱
re.Are.ASCII\w \d 限定为 ASCII，防宽字节/Unicode 绕过过滤混入全角/零宽字符的绕过 Payload

3.url编码实现( urllib.parse)库:

函数名作用示例用法说明
urllib.parse.quote()编码字符串（常用于路径）quote("a b/c") → 'a%20b%2Fc'默认 / 不编码，可改 safe='' 强编码
urllib.parse.quote_plus()编码字符串，并将空格转为 +quote_plus("a b/c") → 'a+b%2Fc'常用于 application/x-www-form-urlencoded
urllib.parse.urlencode()编码字典为查询参数urlencode({'q':'ctf show','page':1}) → 'q=ctf+show&page=1'用于 GET 请求参数拼接
